# RapidoRental 
# 🚖 Rapido Booking System (Java Console Project)

A simple **Rapido Ride Booking System** developed using **Core Java** and **File Handling**. This project allows users to register, login, book rides, view bookings, and cancel bookings through a console-based interface.

---

## 📌 Features

✅ User Registration

✅ User Login Authentication

✅ Ride Booking

✅ Vehicle Suggestions Based on Distance

✅ Fare Calculation

✅ Estimated Travel Time

✅ View User Bookings

✅ Cancel Booking Using Booking ID

✅ File Handling for Data Storage

✅ Logout Functionality

---

## 🛠️ Technologies Used

* ☕ Java
* 📂 File Handling
* 🖥️ Console-Based User Interface
* 📝 Object-Oriented Programming (OOP)

## 🚀 How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Rapido-Booking-System.git
```

### 2️⃣ Open Project Folder

```bash
cd Rapido-Booking-System
```

### 3️⃣ Compile the Java Program

```bash
javac MainApp.java
```

### 4️⃣ Run the Program

```bash
java MainApp
```
